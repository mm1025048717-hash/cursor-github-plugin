✅ 编译已完成，可以直接启动！

📋 启动步骤（3步即可）：
─────────────────────────────────────────────────────────

1️⃣ 在 Cursor 中打开插件项目
   • 文件 → 打开文件夹
   • 选择：cursor-github-plugin 文件夹

2️⃣ 按 F5 启动调试
   • 会打开一个新的 Cursor 窗口
   • 标题栏显示 [Extension Development Host]

3️⃣ 在新窗口中打开助手
   • 按 Ctrl+Shift+P
   • 输入：GitHub AI: 打开助手
   • 回车即可！

─────────────────────────────────────────────────────────

⚙️ 首次使用需要配置 API Key：
─────────────────────────────────────────────────────────
1. 在新窗口中按 Ctrl+,
2. 搜索：githubAI.deepseekApiKey
3. 输入你的 DeepSeek API Key
   （获取地址：https://platform.deepseek.com/api_keys）

─────────────────────────────────────────────────────────
