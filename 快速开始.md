# 🚀 快速开始 - GitHub AI 助手

## ✨ 新特性

现在你只需要**一个对话界面**，用自然语言就能完成所有操作！完全接入 DeepSeek API，极简交互，苹果风格设计。

## 📦 安装与运行

### 1. 确保依赖已安装
```bash
cd cursor-github-plugin
npm install
npm run compile
```

### 2. 在 Cursor 中运行
1. 在 Cursor 中打开 `cursor-github-plugin` 文件夹
2. 按 `F5` 启动调试
3. 会打开一个新的 Cursor 窗口，插件已加载

### 3. 配置 DeepSeek API Key（必需）
1. 在新窗口中按 `Ctrl+,` 打开设置
2. 搜索 "GitHub AI"
3. 输入你的 **DeepSeek API Key**
   - 获取地址：https://www.deepseek.com/
   - 注册账号并获取 API Key

## 🎯 开始使用

### 打开助手
按 `Ctrl+Shift+P`，输入 **"GitHub AI: 打开助手"** 或 **"GitHub AI"**

就会出现一个漂亮的对话界面！

## ⚙️ 配置选项

在设置中搜索 "GitHub AI"：

- **deepseekApiKey** (必需): DeepSeek API Key
- **githubToken** (可选): GitHub Token，提高搜索速率限制
- **downloadPath**: 项目下载路径，默认 `${workspaceFolder}/github-projects`

## 🐛 故障排除

**编译失败？**
```bash
rm -rf node_modules out
npm install
npm run compile
```

**API 调用失败？**
- 检查 DeepSeek API Key 是否正确
- 检查网络连接
- 确保账户有余额

**找不到命令？**
- 确保插件已正确编译（有 `out/extension.js`）
- 重启 Cursor
- 检查扩展是否已激活
