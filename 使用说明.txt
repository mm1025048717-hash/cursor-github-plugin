✨ 新功能：统一对话界面，只需要说话就能完成所有操作！

📦 安装步骤
─────────────────────────────────────────────────────────
1. 安装依赖：cd cursor-github-plugin && npm install
2. 编译：npm run compile
3. 在 Cursor 中按 F5 启动调试
4. 配置 DeepSeek API Key（必需）

🎯 使用方法
─────────────────────────────────────────────────────────
1. 按 Ctrl+Shift+P，输入 "GitHub AI: 打开助手"
2. 在对话界面直接说你想做什么

💬 使用示例
─────────────────────────────────────────────────────────

搜索项目：
  "找一个 React 的待办事项应用"
  "搜索 Python 的机器学习库"

下载项目：
  "下载第1个"
  "下载 react-todo"

修改代码（先打开代码文件）：
  "添加错误处理"
  "优化这个函数"
  "添加注释"

解释代码（先打开代码文件）：
  "这段代码什么意思"
  "解释这个函数"

管理项目：
  "显示我的项目"
  "打开 react-todo"

🎨 设计特点
─────────────────────────────────────────────────────────
✓ 苹果风格：毛玻璃效果、圆角、流畅动画
