## 🚀 如何使用

### 1. 启动项目

```bash
cd cursor-github-plugin
npm install      # 已完成
npm run compile  # 已完成
```

### 2. 在 Cursor 中运行

1. 在 Cursor 中打开 `cursor-github-plugin` 文件夹
2. 按 `F5` 启动调试
3. 会打开新的 Cursor 窗口，插件已加载

### 3. 配置 API Key

1. 按 `Ctrl+,` 打开设置
2. 搜索 "GitHub AI"
3. 输入 **DeepSeek API Key**（必需）
   - 获取：https://www.deepseek.com/

### 4. 开始使用

按 `Ctrl+Shift+P`，输入 **"GitHub AI: 打开助手"**

## 🔧 技术实现

### AI 意图理解流程

```
用户输入自然语言
  ↓
DeepSeek API 理解意图
  ↓
返回 JSON（intent + parameters）
  ↓
根据意图执行相应操作
  ↓
返回结果给用户
```

### 支持的意图类型

- `search` - 搜索项目
- `download` - 下载项目
- `modify` - 修改代码
- `explain` - 解释代码
- `open` - 打开项目
- `list` - 列出项目
- `chat` - 普通对话
