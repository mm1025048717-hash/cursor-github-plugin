# 🔍 搜索 GitHub 项目 - 快速指南

## 🚀 使用我们创建的 Cursor 插件

### 第一步：启动插件

1. **在 Cursor 中打开插件项目**
   - 打开文件夹：`cursor-github-plugin`
   - 确保这是工作区根目录

2. **启动调试**
   - 按 `F5` 键
   - 或点击左侧调试图标 → 选择 "Run Extension"
   - 会打开新的 Cursor 窗口

3. **配置 API Key**（首次使用）
   - 在新窗口中按 `Ctrl+,` 打开设置
   - 搜索：`githubAI.deepseekApiKey`
   - 输入你的 DeepSeek API Key

### 第二步：打开助手

在新窗口中：
1. 按 `Ctrl+Shift+P`
2. 输入：`GitHub AI: 打开助手`
3. 回车

### 第三步：搜索项目

在对话界面中，直接用自然语言说：

**示例：**
```
你: 找一个 React 的待办事项应用
AI: 🔍 找到 1250 个项目，这里是最相关的 5 个：
    1. ⭐ facebook/react-todo (15.2k stars)
    2. ⭐ microsoft/todo-app (8.5k stars)
    ...

你: 下载第1个
AI: 📥 正在下载...
```

## ⚡ 快速命令

如果不想打开对话界面，也可以直接使用命令：

1. 按 `Ctrl+Shift+P`
2. 输入：`GitHub AI: 搜索项目`
3. 输入你的搜索需求
